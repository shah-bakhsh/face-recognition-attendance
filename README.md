# Face Recognition Attendance System

A Python-based attendance system that uses **face recognition** to automatically register and track student attendance. Built with **OpenCV**, **face_recognition**, and **Pandas**, this project provides an easy way to mark attendance in real-time.


## Features

- Register new students with multiple face captures.
- Real-time face recognition using webcam.
- Automatically logs attendance with timestamp.
- Stores attendance records in a CSV file.
- Handles multiple students and unknown faces.
- User-friendly command-line interface.



## Technologies & Libraries

- Python 3.x
- [OpenCV](https://opencv.org/)
- [face_recognition](https://github.com/ageitgey/face_recognition)
- [Pandas](https://pandas.pydata.org/)
- NumPy
- Pickle (for storing face encodings)
